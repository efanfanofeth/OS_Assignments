1. Add all java and txt files to the same workplace

2. Run the Driver.java file.

3. In the terminal, write "java Driver [algorithm] [schedule]

4. For algorithm choose between FCFS, PRI, or RR, for schedule write the txt file you would like to use

5. Terminal will run the algorithm with the txt file of the users choice, the user will then be prompted again to choose the algorithm and schedule indefinitely until the program is closed
